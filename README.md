# Arnav's Game Macropad

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)
![Made with KiCad](https://img.shields.io/badge/Made%20with-KiCad-blue)
![Built with Fusion 360](https://img.shields.io/badge/CAD-Fusion%20360-orange)

Submission for Hackpad.

This is a macropad built for playing games. It has an OLED screen, allowing smaller games to be shown. There are 6 buttons, which can be used as the 4 directions and then 2 more for other actions.

Features:

- OLED Screen
- 6 Buttons
- SK6812 Mini LED

BOM:

- 0.91in OLED Screen
- 6x Cherry MX Switches
- 6x Blank DSA Keycaps
- SK6812 MINI LED
- XIAO RP2040
- 3d Printed Case (Top and Bottom)
- 4x Bolts

## Assembly

Assembling will not take very long. Once the PCB is completed (the OLED screen, XIAO, and buttons are all connected), it should be able to be put into the case. Then you can put the top of the case on, and it's done.
